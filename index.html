Data Management

Data Formatting

Data must be entered as comma, tab, or semicolon separated values using the following column order:

Pathway, Step, Label, Energy, [Options]
  • Pathway: Name of the reaction path (e.g., Main or Alt-A). Points sharing the same pathway name will be connected by lines.
  • Step: Integer representing the horizontal reaction coordinate (e.g., 1, 2, 3). Shared step numbers across different pathways will align vertically.
  • Label: The text to display for the species (supports subscripts and superscripts).
  • Energy: The numerical energy value in your chosen units (e.g., 32.4 or -5.2).

The 5th Column [Options]

You can add an optional 5th column to precisely control how labels and numbers are displayed for individual points.

  • #L — Hides the text label for this point
  • #E — Hides the numerical energy value for this point
  • Position shifting (Y@X format): Adjusts vertical and horizontal positions of labels independently
    • 10 — Shifts label UP by 10 energy units
    • 10@0.2 — Shifts label UP by 10 units AND RIGHT by 0.2 steps
    • @-0.3 — Shifts label LEFT by 0.3 steps (no vertical shift)
Path1, 3, INT1, -5.20, #E 10@0.2

This example hides the energy number and shifts the "INT1" label UP and slightly RIGHT.

Subscripts & Superscripts

Format chemical formulas and transition state symbols in both the Data Tab and Custom Annotations Tab:

  • Subscripts: Wrap text in _{...}
    Example: H_{2}O renders as H2O
  • Superscripts: Wrap text in ^{...}
    Example: E^{‡} renders as E‡

Barrier Markers

Barrier Markers automatically calculate the exact energy difference (Final − Initial) between two steps on the same pathway. If overlapping elements occur, use the X-Offset input to manually nudge the barrier left (e.g., -0.2) or right (e.g., 0.2).

Custom Annotations

Custom Annotations allow you to place free-floating text or arrows anywhere on the canvas. X and Y coordinates map directly to the plot axes, making precise positioning straightforward.

Keyboard Shortcuts

  • Ctrl/Cmd + Enter — Redraw diagram from any tab

Export Recommendations

For publications and presentations: Export as SVG for best quality and editability. SVG files are vector-based, resolution-independent, and can be further refined in Adobe Illustrator, Inkscape, or similar software.

For quick sharing: PNG exports work well for slides and documents where raster images are acceptable.
